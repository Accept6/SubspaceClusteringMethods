This MATLAB code is for the clustering method published in the following paper:
Baohua Li, Ying Zhang, Zhouchen Lin, and Huchuan Lu, Subspace Clustering by Mixture of Gaussian Regression, IEEE Conference on Computer Vision and Pattern Recognition (CVPR 2015).

The main function is maincode.m.

This is the second version of code. We appreciate any comments/suggestions. Feel free to ask us any questions, please contact us.
